About

Drupug is a tool which helps you to find always-up-to-date information about the drugs you are taking. This tool was created as part of the HACK Zurich 2021 Event.

Example 1: Xyosted

Xyosten is medication which is used mainly in the treatment of low testosterone levels in men.
Example 2: Zubsolv

Zubsolv is used to treat opioid use disorder, and reduces the mortality of opioid use disorder by 50%.

About

Drupug is a tool which helps you to find always-up-to-date information about the drugs you are taking. This tool was created as part of the HACK Zurich 2021 Event.

Example 1: Janumet

Janumet is a fixed-dose combination anti-diabetic medication used to treat type 2 diabetes.
Example 2: Hydrea

Hydrea is a medication used in sickle-cell disease, essential thrombocythemia, chronic myelogenous leukemia and cervical cancer.
